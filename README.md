# Минимальный React-проект

Для локально разработки:

1. Установите [git](https://git-scm.com/install/) и [NodeJS](https://nodejs.org/en/download)
2. Запустите на своем компьютере команду в папке с вашими проектами `git clone https://github.com/Opty1712/mini-store.git`, в результате появится папка `mini-store`
3. Перейдите в папку проекта `cd mini-store`
4. Установите зависимости `npm install`
5. Запустите проект `npm run dev`
